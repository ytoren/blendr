# Blend ML
This is an R library containig function that allow the *blending* of machine learning (ML) insights into parametirc regression (typicaly GLM).

## Setup
```{r installation} 
# install.packages("devtools")
library(devtools)
devtools::install_github("ytoren/blendML")
```
